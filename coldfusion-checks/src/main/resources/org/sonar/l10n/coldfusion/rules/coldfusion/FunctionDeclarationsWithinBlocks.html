Do not do this : 

if (x) {
  function foo() {}
}

While most script engines support Function Declarations within blocks it is not part of CF. Worse implementations are inconsistent with each other. CF only allows for Function Declarations in the root statement list of a script or function. Instead use a variable initialized with a Function Expression to define a function within a block : 

if (x) {
  var foo = function() {}
}
